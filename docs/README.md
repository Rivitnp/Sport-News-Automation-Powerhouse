## Key Information

### Target Audience
- Nepal and India sports betting enthusiasts
- Focus: Cricket and Football (NOT American sports)

### Publishing Schedule
- Every 3 hours (8 runs per day)
- 10 articles per run
- 80 articles per day total

### Traffic Goal
- Minimum: 10,000 visitors/month
- Expected: 48,000-72,000 visitors/month

### Key Features
- ✅ Automatic betting tips with 1xbet.com mentions
- ✅ 18+ disclaimers on all betting content
- ✅ Copyright-free AI-generated images
- ✅ SEO optimization with Google Analytics
- ✅ Cricket/Football priority filtering

### Technology Stack
- Python 3.11
- GitHub Actions (automation)
- WordPress REST API (publishing)
- OpenRouter AI (content generation)
- Cloudflare Flux (image generation)
- Serper API (trending keywords)
